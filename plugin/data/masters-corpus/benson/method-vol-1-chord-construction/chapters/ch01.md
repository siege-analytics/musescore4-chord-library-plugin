# Chapter 1 — Staff, Clefs & Musical Notes

_Pages 20-25._

## Interval definition

> A musical interval is the distance from one note (sound) to another. They may be described
>     as perfect, major, minor, diminished or augmented. The smallest existing interval is called
>     half step, which can also be called half tone or semitone; two half steps form one whole step,
>     which can be called whole tone.

**Significance (p. 20).** Defines interval and the half-step/whole-step vocabulary the rest of the book builds on.

## Natural notes and letters

> There
>     are 12 existing notes in western music, but actually we only need 7 of them for such
>     representation. We will call them natural notes: Do, Re, Mi, Fa, Sol, La, Ti. Each one of these
>     notes are represented by the 7 Jirst letters from the alphabet:
>
>       • Do = C / Re = D / Mi = E / Fa = F / Sol = G / La = A / Ti = B

**Significance (p. 20).** Establishes the 12-note system, the 7 naturals, and the solfege-to-letter mapping used throughout.

## Accidentals derive remaining notes

> The other 5 notes will be derived from these main 7, using what we call music accidentals,
>     which can rise or lower each note one half step.

**Significance (p. 20).** Explains why only 7 letter names cover 12 pitches — accidentals do the rest.

## Staff definition

> Staff or Stave is the set of 5 horizontal, parallel and equidistant lines which create four spaces
> between themselves. Each line and each space represent a different musical pitch.

**Significance (p. 21).** Defines the staff's geometry — the basic surface on which all notation lives.

## Clef purpose

> The Clef is a symbol that we place on the staff to indicate in which line or space the musical
> notes should be placed.

**Significance (p. 21).** Defines what a clef does — anchors note names to staff positions.
