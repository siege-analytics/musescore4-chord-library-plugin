## altered extensions usage rule

> These chords can be used over any dominant 7th chord in your comping to create
> extra tension, but if you’re new to these sounds start with using them in minor keys.

**Significance (p. 35).** This is the primary usage rule for altered dominant extensions, giving players a concrete starting point and a broader principle for deployment.

## altered extensions in minor ii-V-I

## #11 vs b5 distinction

> A lot of people are confused about the difference between extensions #11 and b5.
> Both are the same note, an f# (=gb) in C7.

**Significance (p. 36).** This defines a commonly misunderstood enharmonic equivalence that underpins the chapter's chord taxonomy.
