# Chapter 3 — Module 3 - Let's Create a Chord Melody Arrangement (First Steps & Harmony Analysis)

_Distillation; backed by `chapters/ch03.md`._

Chapter 3 walks the student through the front half of an actual arrangement workflow — melody fingering, comping, and harmonic analysis — before any voicing selection happens. It reaffirms the top-2-string canvas with an explicit carve-out that occasional dips to the 3rd or 4th string are fine (Top-2-string melody constraint), insists the arrangement be notated on paper or software so it can be iteratively refined (Write down the arrangement), and prescribes a pre-step of simply comping the song's basic chords in a band context to train the ear before voicing choices begin (Basic comping before voicing). The harmonic analysis pass has three explicit outputs: chord qualities (major/minor/dominant), key changes, and recognizable progressions like ii-V-I (Harmonic analysis goals), notated using uppercase Roman numerals for major/dominant and lowercase for minor (Roman numeral conventions). The linchpin of the whole 'painting by numbers' method is then introduced: knowing each melody note's interval relative to its underlying chord is what enables dictionary lookup of a voicing with that interval on top (Why interval analysis). O'Rourke flags the most common beginner mistake — labeling intervals against the overall key rather than the current chord (Interval relates to current chord) — and reinforces this as his headline 'golden rule' (Golden rule of interval analysis), backed by the rule that the reference is always the major scale of the current chord's root (Major scale reference rule). Load-bearing terminology includes the closed chord-tone vocabulary used as dictionary index keys — 1, b9, 9, b3, 3, 11, #11, 5, b13, 13, b7, 7 (Chord-tone vocabulary) — plus the PT (passing tone) escape hatch for unharmonized chromatic notes (Passing-tone label) and the edge case where a melody note anticipating the next chord is labeled against that next chord (Melody anticipates next chord).
