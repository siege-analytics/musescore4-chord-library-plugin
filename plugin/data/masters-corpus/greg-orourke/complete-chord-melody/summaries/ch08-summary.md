# Chapter 8 — Appendix 1 - Chord Dictionaries

_Distillation; backed by `chapters/ch08.md`._

Chapter 8 is a reference appendix rather than instructional material, consolidating the chord voicings introduced earlier in the method into a single chord-melody dictionary keyed to Module 3 (Chord melody dictionary purpose). The load-bearing organizing taxonomy is a four-way split of ii V i voicings by quality and string set: major ii V I on string set 4321 and on string set 5432 (Major II V I string set 4321, Major II V I string set 5432), and minor ii V i on the same two string sets with the b5 and b9 alterations characteristic of the minor cadence (Minor II V I string set 4321, Minor II V I string set 5432). A second section catalogues altered/extended dominants as the method's color resource (Tensions and extensions on dominants) and then pivots to the solo-guitar voicing reference for Module 4 (Solo guitar shell voicings), built on the structural backbone of 6th-string-root shell voicings (Shell voicings root on 6th) plus a catalog of extensions stacked on top of those shells (Shell voicing extensions catalog)—the load-bearing harmonic vocabulary O'Rourke uses for solo arranging.
