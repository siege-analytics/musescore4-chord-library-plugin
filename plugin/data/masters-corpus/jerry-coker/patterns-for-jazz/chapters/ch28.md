## Augmented scale intervals and spelling

> The augmented scale, like the diminished scale, is a symmetrical scale. This time the intervals al-
> ternate between half-sieps and one and one-half steps (minor thirds or augmented seconds). On C
> the scale would be spelled C, D-sharp, E, G, A-flat, B, C. Note that there are, because of the wide
> augmented second intervals, only six tones in the scale.

**Significance (p. 134).** This is the foundational definition of the augmented scale, giving its interval structure, spelling, and the reason it has only six tones — the core theoretical anchor for everything that follows.

## Only four augmented scales exist

> Only four (chro-
> matically adjacent) augmented scales exist.

**Significance (p. 134).** This rule establishes the symmetry property that limits the augmented scale to four distinct transpositions, a critical fact for knowing which scales to practice and how they overlap.

## Augmented scale derived from augmented triads

> As its name implies. the augmented scale is derived from the use of augmented triads. If we examine
> the above spelling of the C augmented scale, we see that it contains cwo augmented triads, chromati-
> cally adjacent: B, D-sharp, G (or F double sharp) and C, E, A-flat (or G-sharp).

**Significance (p. 134).** Explains the harmonic origin of the scale by showing it contains two chromatically adjacent augmented triads, giving the student both a mnemonic and a chord-scale relationship to exploit.

## Chord-scale fit: M7+5 and M7

> In addition to fitting augmented triads, the augmented scale fits a rarely-used chord, the major
> seventh chord with an augmented fifth (M7+5)°", Again, since this chord is relatively new, like the
> scale, it could enjoy wider use in the future, because both the chord and the scale present interesting
> sounds that might prove attractive to the jazz improviser. With slightly less effect, the scale can be
> applied to the major seventh chord (M7).

**Significance (p. 134).** Provides the definitive chord-scale mapping — M7+5 as the primary target chord and M7 as a secondary option — which tells the improviser exactly when to deploy this scale.
