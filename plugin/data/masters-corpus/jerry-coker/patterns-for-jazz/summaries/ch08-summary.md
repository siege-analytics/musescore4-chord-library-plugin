# Chapter 8 — Diatonic Chords

_Distillation; backed by `chapters/ch08.md`._

Chapter 8 introduces diatonic chords using Roman numeral chord notation, the foundational system for designating triads built on each scale tone. The chapter establishes a universal principle that applies to all major scales: triads can be constructed on every tone of any major scale, with the student expected to write out scale-tone triads for all twelve keys as a prerequisite before moving to pattern work. The load-bearing terminology here is roman numerals (as distinct from arabic numerals used for scale degrees) and scale-tone triads. Rather than offering new techniques, this chapter is fundamentally conceptual and prerequisite-oriented, teaching the notational system and structural principle that will underpin all subsequent chord work in the method.
