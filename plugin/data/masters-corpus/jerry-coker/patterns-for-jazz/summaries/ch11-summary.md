# Chapter 11 — Minor Triads

_Distillation; backed by `chapters/ch11.md`._

Chapter 11 introduces minor triads and related chords by deriving them from the B Dorian Mode using numerical formulas, establishing a unified system where all minor chord types (Minor Triad, Minor Sixth, Minor Seventh, and Minor Ninth) are constructed by extracting specific tone numbers from the same modal foundation. The foundational Minor Triad formula extracts tones 1-3-5 from the Dorian Mode, while extensions like the Minor Sixth Chord add tone 6 (1-3-5-6), demonstrating how each chord type builds systematically from the basic triad. The chapter introduces the Dorian Mode octave number system, which labels tones 1–7 in the first octave and continues the numbering across subsequent octaves to indicate distances from the starting tone, enabling correct interpretation of any formula. This modal-derivation approach—reducing all minor chord construction to extractable numerical patterns—is characteristic of the method's pedagogical efficiency: one mode, multiple formulas, complete minor harmony coverage.
