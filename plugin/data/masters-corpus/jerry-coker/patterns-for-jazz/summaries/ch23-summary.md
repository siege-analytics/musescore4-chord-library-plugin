# Chapter 23 — Turnarounds (Turnbacks)

_Distillation; backed by `chapters/ch23.md`._

Chapter 23 introduces turnarounds (turnbacks) as a harmonic technique based on stationary-root chord-type permutation, where the root remains fixed across successive chords while the chord type and function change. The chapter establishes this concept by contrasting it against conventional root-motion patterns such as cycle-of-fifths and chromatic descent progressions. The core mechanic exploits chord-type permutation—changing from major-seventh to minor or dominant qualities on the same root—to shift both harmonic function and key center simultaneously, creating the characteristic turnback sound. This technique differs fundamentally from traditional progressions where root motion drives harmonic change; instead, stationary-root permutation generates movement through quality variation alone. The patterns explored focus on this root-held structure as the defining characteristic of effective turnaround writing.
