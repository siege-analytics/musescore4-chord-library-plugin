# Chapter 28 — Augmented Scale

_Distillation; backed by `chapters/ch28.md`._

Chapter 28 introduces the augmented scale, a symmetrical six-tone scale alternating between half-steps and augmented seconds (minor thirds), spelled on C as C-D#-E-G-Ab-B. Only four chromatically adjacent augmented scales exist due to this symmetrical structure. The augmented scale derives from augmented triads: the C augmented scale contains two chromatically adjacent augmented triads (B-D#-G and C-E-Ab), establishing the chord-scale relationship. Its primary harmonic application is the M7+5 chord (major seventh with augmented fifth), with secondary use over M7; both chords provide the harmonic context for deploying this scale in jazz improvisation. Beyond chord-specific contexts, the augmented scale's symmetry and "mystical, keyless sound" position it as a free-form device for atonal or non-functional applications, expanding the improviser's palette beyond traditional harmonic frameworks.
