## Open Position Chords requirement

> You need to know the basic open-position chords as well as the standard barre chord
> fingerings. Changing from one chord to another should not be a problem. If you don’t
> know these chords, learn them well before proceeding.

**Significance (p. 6).** This establishes the prerequisite technical baseline the entire series assumes, making it the entry gate for all subsequent instruction.

## Standard notation: pitch and rhythm

> Reading standard music notation is a necessary skill for anyone interested in learning to
> play jazz. Once you get the idea, you'll find that it’s really easy, and then a whole world of
> instructional books and great music will open up for you. There are two basic elements to
> standard notation: pitch and rhythm. Every note has a particular note name (pitch) and
> particular duration (rhythm). The line or space on a note falls on tells you the pitch.

**Significance (p. 8).** This frames standard notation as a non-negotiable skill for jazz and defines its two fundamental dimensions, establishing why the book uses it throughout.

## Time signature definition

> Every piece has numbers at the beginning, called the time signature, that tell us how to
> count the time. The top number represents the number of beats or counts per measure.
> The bottom number represents the type of note receiving one count. The most common
> time signature, 4 is shown below. In 4 time, there are four beats per measure, and the
> quarter note (d) receives one beat.

**Significance (p. 8).** This defines the time signature concept and decodes the top/bottom number meanings, which are necessary for reading every notated example in the series.

## Note values and rests

> The appearance of the note—the type of note head or stem that it has—tells you the
> rhythm. Rests tell you when and how long not to play, which is also an important aspect of
> rhythm.

**Significance (p. 8).** This rule that rests are as important as notes frames silence as a rhythmic element, a key interpretive concept for jazz phrasing.

## Progression / changes terminology

> You need to be familiar with these basic progressions. Progression refers to the movement

## Strumming coordination requirement

> You will need to know some basic strumming patterns as well. The patterns themselves
> are not that important. The idea is that you should have the coordination to strum a
> pattern while changing chords. Read the section on “Basic Strumming Technique” on page
> 87 for some tips.

**Significance (p. 9).** This defines the minimum rhythmic-coordination prerequisite (maintaining a pattern while changing chords) and cross-references where technique tips live.
