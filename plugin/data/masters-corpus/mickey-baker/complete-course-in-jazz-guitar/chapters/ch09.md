## Chromatic fingering invariance

> *You are to use the same fingering, up and down the fingerboard.

**Significance (p. 30).** States the load-bearing position-system rule: a single fingering shape for a run is transposed by simple position-shift, which is how Baker turns one learned pattern into a 12-key vocabulary.

## Same fingering all directions

> use the same fingering going up and down the fingerboard.

**Significance (p. 31).** Reiterates the invariant-fingering doctrine for dominant 7th/9th runs — the mechanical key to executing Baker's chromatic transposition drill at tempo.
