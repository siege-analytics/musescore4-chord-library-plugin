# Chapter 5 — Modal Interchange

_Distillation; backed by `chapters/ch05.md`._

Chapter 5 introduces modal interchange as a reharmonization technique where chords derived from parallel scales (natural minor, harmonic minor, melodic minor, Dorian) substitute for diatonic chords in a major key, with melody compatibility as the governing constraint. The chapter's chapter-file has no extracted load-bearing prose passages due to a validation issue during quote extraction; however, the chapter known content covers twelve common modal interchange chords (such as I-7, I-6, I-(Maj7), bIIMaj7, IV-, V-7, bVIIMaj7) and the rule that minor-quality MI chords clash with melodies containing the major third (creating b9 intervals). Modal interchange chords function as links between diatonic chords via strong root motion into a target, where new chord tones produce listener interest. A three-chord limit is recommended before returning to diatonic harmony to avoid an ambiguous tonal center. The MI substitution does not need to preserve functional category (tonic/subdominant/dominant), distinguishing it from simple substitution which keeps function but yields less color.
