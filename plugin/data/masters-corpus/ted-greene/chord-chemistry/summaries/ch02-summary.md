# Chapter 2 — Chord Reference Charts Explanation

_Distillation; backed by `chapters/ch02.md`._

Chapter 2 provides foundational reference material for navigating the guitar neck and reading chord diagrams. It introduces the core neck-memorization system built on two sets of reference points—the 1st reference points at the 5th fret (4th fret for the 3rd string) that match the open strings one octave higher, and the 2nd reference points at the 12th fret that duplicate the open strings an octave up—along with the 1st and 6th string symmetry rule that halves memorization load. The chapter establishes essential terminology and conventions: open string, enharmonic equivalence (Gb and F#), moveable chords, and the silent/deadened strings rule for chord voicing. It also codifies the book's notation system, including finger numbering, the 1st finger split-fret technique, and chord symbol conventions (slash for major type, + for sharpened 5th, sus for 4th replacing 3rd). The chapter prescribes a systematic 15-minute daily practice to internalize the fingerboard and defines the term 'TYPE' to categorize related chord families.
