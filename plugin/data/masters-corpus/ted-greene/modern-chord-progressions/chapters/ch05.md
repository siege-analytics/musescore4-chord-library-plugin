## Building a chord scale by stepwise voice motion

> a chord scale can be built by starting on a major chord and MOVING EACH NOTE IN THE CHORD UP
> TO THE NEXT NOTE IN THE MAJOR SCALE WHICH HAS THE SAME NAME AS THE CHORD. For instance, in
> the above example, the C chord contains the following notes (from the bass up): EC GC; to find the next diatonic
> chord in the chord scale, the E moves up to F, the C to D, the G to A, and the C to D, thereby forming the Dm chord.
> If you continue the same procedurc, the notes of the Dm would move up (in the C major scale) to form the Em chord
> and these in turn would move up to form the F chord and so on.

**Significance (p. 17).** Precise voice-leading algorithm for constructing any diatonic chord scale.
