# Chapter 3 — Homonyms

_Distillation; backed by `chapters/ch03.md`._

This chapter teaches three techniques for memorizing chord forms — shape-based, interval-based, and root-based anchoring — with the root-anchor technique requiring prerequisite knowledge of note names on the fingerboard, supported by concrete drill examples and a rule that experienced players combine all three methods eclectically. The chapter's central concept is chord homonyms: chords that sound identical but receive different names depending on harmonic context (e.g., the same form becomes Em6, C#m7b5, or A9 based on the progression), governed by a "common sense" rule that context determines naming. Load-bearing terminology introduced includes homonym, diatonic (using only notes of the scale), analyze (identifying present chord tones), and voice leading (connecting chord notes smoothly across progressions), with a foundational reference table showing how chords are constructed by stacking thirds (triads as 1-3-5, seventh chords as 1-3-5-7, etc.). The chapter emphasizes that mastery requires learning all useful names for each chord form to avoid long-term difficulty.
