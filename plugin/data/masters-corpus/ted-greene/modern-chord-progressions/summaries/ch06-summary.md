# Chapter 6 — Progressions Built from Scalewise Root Movements and Progressions Built from Scalewise Bass Lines (Major Key)

_Distillation; backed by `chapters/ch06.md`._

This chapter introduces two foundational techniques for constructing progressions: Scalewise Root Movements (SWR), which derive from diatonic chord scales, and Scalewise Bass Lines (SWB), which require understanding musical inversions to construct effectively. The chapter instructs readers to build diatonic 7th chord forms and emphasizes the critical rule that chord scales must be practiced descending as well as ascending. A key voicing technique presented is using chord decoration or voicing adjustments to soften dissonance, making progressions more cohesive. While the author notes that SWB applications will be covered thoroughly in Volume 2, these brief examples serve as foundational "teasers" for understanding how scalewise bass progressions create musically attractive sounds.
