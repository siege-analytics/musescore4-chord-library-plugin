# Chapter 11 — III7 VI7 II7 V7

_Distillation; backed by `chapters/ch11.md`._

Chapter 11 introduces the III7 VI7 II7 V7 harmonic progression, the final major progression covered in this book, organized by which chord initiates the sequence. The chapter teaches that altered dominant 7th chords — those with characteristic extensions like #9s, b9s, #5s, and b5s — create distinctly "spicy" sounds within this progression, and provides substitution rules allowing iii7 for III7 types and vi7 for VI7 types to expand harmonic vocabulary. Key compositional techniques include combining altered progressions with more diatonic "major scaley" progressions for ear training, remembering that V7 can always resolve to I, and using contextual substitutions like A7b5 in place of itself when progressing to D7b9b5. The master's characteristic move throughout is pacing altered-chord introduction by first preparing the ear with normal diatonic sounds, and the load-bearing terminology includes altered dominants, diatonic preparation, and consonant/dissonant progression contrast as a compositional principle.
