# The Method for Guitar (1939) — Statement of Outputs

## Overview

George Van Eps's *The Method for Guitar* (1939) is a three-chapter technical curriculum that reduces guitar playing to an explicit set of mechanical laws, each stated as a near-absolute rule with justification drawn from biomechanics and physics rather than from feel or tradition. The book's central argument is that guitar technique is not intuitive but legislative: posture, hand position, finger behavior, pick motion, and key-traversal are all governed by rules that, once internalized at whatever tempo permits perfect execution, produce a reliable and transferable technique. The method is also inseparable from music theory — its exercises are not physical drills decorated with chord labels but harmonized scale studies in which voice leading, chord structure, and key relationship are absorbed simultaneously with the physical technique.

The three chapters form a tightly sequenced architecture. Chapter 1 establishes the biomechanical substrate — seated position, guitar angle, right-hand oscillation mechanics, left-hand thumb and finger placement, the dynamic-axis rule, the attack-and-exit timing principle, slow-practice tempo, and the Solfeggio all-keys framework — that every subsequent chapter presupposes without re-stating. Chapter 2 applies that substrate to harmonized major scale triads in six distinct fingering forms, introducing arpeggio picking, legato shift, finger flattening, and the sequence-form traversal pattern. Chapter 3 applies the same architecture to harmonized minor scale triads and their diminished and open-voicing variants, adding the rocker-arm technique, contrary motion, diminished symmetry, and the etude composition method as the mechanism for internalizing principles beyond printed exercises.

Taken together, the three chapters constitute a unified system: the posture and grip laws from Chapter 1 are the constant substrate, the six-form harmonized scale is the primary technical vehicle, and arpeggio picking, legato shift, finger flattening, and rocker-arm technique are the named moves through which that vehicle is mastered. The Solfeggio transposition rule is the governing rule that ensures no key is privileged and no exercise is ever truly complete until it has been executed equally in all twelve.

## Systems

### Six-Form Harmonized Scale System

The central technical vehicle of the 1939 method is a set of six distinct fingering forms that partition the guitar neck into regions, each assigned to specific string sets and key ranges. The forms are not interchangeable shortcuts for the same content: each occupies a defined fingerboard region, and the method requires all six to be practiced independently and equally — in long, medium, and condensed variants — to build what Chapter 2 explicitly calls "balanced technique." The system is first applied to major triads (Chapter 2) and then carried forward to minor triads, broken string sets, open voicings, and diminished variants (Chapter 3). All six forms are practiced in all twelve keys via the Solfeggio transposition rule.

**Members:**
- **First Form** — second and first string sets; long form; key range C up to F
- **Second Form** — entirely on the second string set; long form; key range C up to C♯ (D if possible)
- **Third Form** — second and first string sets with a different crossover scale degree than Form 1; long form; key range C up to F
- **Fourth Form** — third, second, and first string sets; long form; key range A♭ up to D♭ (D if possible)
- **Fifth Form** — condensed form; key range A♭ up to E
- **Sixth Form** — most condensed form; key range F♯ up to C♯
- **Harmonized Minor Scale Forms** — six parallel forms applied to the harmonized minor scale, including broken string sets, open voicings, and diminished variants

**Traversal rules:**

- **Legato Shift Between Forms** (`PositionContinuity`): Position changes between forms must be executed without sliding, with a momentary pressure release and the next formation pre-built during the hand's motion. "Do not slide the left hand in the modulation. It should be a fast, accurate change." The underlying legato principle — "changes from formation to formation must be executed in the least amount of time" — is stated in Chapter 1 and enforced in Chapter 2. *(Chapter 1 summary; Chapter 2 summary)*

- **String-Set Crossover Between Forms** (`StringSetTransition`): When moving between forms that cross string sets, the octave jump must be executed cleanly. "Crossing over the sets of strings to jump the octave presents a different problem as this must be done cleanly." *(Chapter 2 summary)*

- **Sequence-Form Traversal Pattern** (`_pending:step-sequence-traversal`): Within each form, the harmonized scale is traversed in a three-steps-up-and-back pattern, ascending then reversing descending. *(Chapter 2 summary)*

- **Diminished Symmetry Four-Fret Jump** (`SymmetryMovement`): Diminished voicings repeat every four frets; traversal uses the same fingering transposed up or down by four frets with fast, slide-free jumps. *(Chapter 3 summary)*

- **Rocker-Arm Legato Transition for Diminished Triads** (`VoiceMotion`): Alternating first and fourth fingers like a rocker arm achieves legato across diminished triad sequences without sliding. *(Chapter 3 summary)*

- **Two-Octave Form Linkage** (`PositionContinuity`): After all six forms are perfected individually, lower and upper forms are joined without a break to produce a seamless two-octave harmonized scale. *(Chapter 2 summary)*

**Modification rules:**

- **Finger Flattening for Added Voice** (`DensityCeiling`): A finger flattens from arched position to sustain an adjacent note simultaneously, functionally adding a fifth voice. Chapter 2 names this the method's most advanced left-hand principle. *(Chapter 2 summary)*

- **Open Voicing via String Deadening** (`OmissionAllow`): A fretting finger simultaneously mutes an adjacent string to produce an open voicing. *(Chapter 3 summary)*

- **Barre-Finger Substitution Across Fingers** (`SubstitutionExpand`): Any barre position is practiced with each finger (1st, 2nd, 3rd, 4th) performing the barre in turn. *(Chapter 2 summary)*

- **Inner-Voice Sustain Under Moving Melody** (`VoiceMotion`): Inner voices must sustain their full written value while the melodic line moves. *(Chapter 2 summary)*

- **Contrary Motion Voice Expansion** (`VoiceMotion`): Two voices are set in contrary motion to stretch the hand's span. *(Chapter 3 summary)*

- **Seventh-Arpeggio Range Truncation** (`OmissionAllow`): Full-fingerboard arpeggio forms exceeding range are modified by omitting the first measure descending or the fifth ascending. *(Chapter 3 summary)*

### Right-Hand Picking Technique System

Van Eps codifies right-hand technique as a rule-governed system with two principal picking modes and one positional-override path. **Arpeggio picking** is a pulsation technique with downstrokes stopping against the adjacent string and upstrokes returning freely; it is the default mode for all harmonized scale studies. **Alternate picking** assigns downstrokes to beats 1 and 3 and upstrokes to beats 2 and 4, prescribed specifically for second and fourth triads in minor contexts. The **arc-path** is a pick-return path modification applied when a sustained voice sits above the target string. Underlying all three is the dynamic-axis rule (wrist axis over the highest note) and the 2:1 elbow-to-hand speed ratio.

**Members:**
- **Arpeggio Picking (Pulsation)** — downstrokes pulsed with a slight kick, stopped against the next-highest string; upstrokes return freely
- **Alternate Picking** — downstrokes on beats 1 and 3, upstrokes on beats 2 and 4 within a triad measure
- **Arc-Path Pick Return (Open Voicing)** — pick travels a small arc around the upper string after a downstroke to avoid deadening a sustaining voice

**Traversal rules:**

- **Picking-Mode Assignment by Exercise Type** (`_pending:mode-assignment`): Arpeggio picking is the default; alternate picking is assigned specifically to beats 2 and 4 in minor triad contexts. *(Chapter 3 summary)*

- **Up-Stroke Retrospective Revision Drill** (`TextureCycle`): All prior down-stroke-only exercises are revisited with up-strokes and then with alternating strokes. *(Chapter 3 summary)*

- **Dynamic Axis Rule** (`_pending:dynamic-axis`): The wrist axis is positioned directly over the highest note being played so the top voice predominates. *(Chapter 1 summary)*

**Modification rules:**

- **Arc-Path Modification for Open-Voicing Contexts** (`_pending:pick-path-modification`): When a sustained voice sits above the target string, the pick-return path is modified from a straight return to a small arc around the upper string. *(Chapter 3 summary)*

### Solfeggio All-Keys Transposition System

The Solfeggio system is the method's key-equality framework and its governing meta-rule. Every exercise is conceived relative to "do" — the tonic of whatever key is being practiced — and this relativization obligates the student to practice each exercise in all twelve keys without privileging any. The system is not a separate exercise block but a constraint applied to every form, every technique, and every exercise across all three chapters. The key-range tables introduced in Chapter 2 define the practical limits within which each form can be transposed before a different form must be selected.

**Members:**
- **Tonic as 'Do'** — each key's tonic is treated as "do," making all twelve keys structurally identical
- **Twelve-Key Practice Cycle** — every exercise is practiced in all twelve keys; key-range tables per form define the practical limits of transposition

**Traversal rules:**

- **Key-Equality Traversal Rule** (`FamilyCoherence`): No key receives privileged practice attention; the student moves through all twelve keys without favoring familiar ones. *(Chapter 1 summary)*

- **Form-Range-Bounded Transposition** (`PositionContinuity`): Each fingering form is transposed only within its prescribed key range; transpositions outside that range require switching to a different form. *(Chapter 2 summary)*

**Modification rules:** None defined for this system.
